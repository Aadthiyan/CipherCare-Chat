# Contributing Guidelines

## Code Style
- **Python**: Follow PEP 8. Use `black` and `isort` for formatting. typings are required.
- **JavaScript/React**: Use Airbnb style guide. Use `Prettier` and `ESLint`.

## Pull Request Process
1.  Create a feature branch: `feature/my-feature`.
2.  Add tests for your changes.
3.  Ensure CI passes.
4.  Submitt a PR with a description of changes.
5.  Request review from at least one team member.

## Commit Messages
Use [Conventional Commits](https://www.conventionalcommits.org/):
- `feat: add new feature`
- `fix: resolve bug`
- `docs: update documentation`
